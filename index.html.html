AI-Powered Startup Evaluation System

Personal Project

Automation system that replaces 2-3 hours of manual startup research with a single trigger

After doing hundreds of startup evaluations manually, I saw the bottleneck and built a solution. The system takes a startup name as input, pulls data from multiple sources, runs it through an LLM-based scoring framework with multiple AI agents handling different parts of the analysis, and generates a structured evaluation report.

How it works: Built on n8n with AI agents orchestrated across research, scoring, and report generation steps. Each agent handles a distinct task and passes structured output to the next. The final output is a formatted evaluation ready for stakeholder review.

~90% Time saved per evaluation
